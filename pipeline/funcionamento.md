# Funcionamento do Processo de ETL

> O processo de ETL (Coleta, Tratamento e Salvamento) consiste em um script automatizado na linguagem de programação PYTHON que realiza requisições de dados em fontes governamentais seguras, realiza tratamentos de dados especifícos e salva em banco de dados.

### Características do Processo
- Realiza coleta assincrona dos dados
- Após coleta, realiza automaticamente o tratamento específico para aquele determinado indicador com base na sua fonte de origem
- Realiza o salvamento em banco de dados
